# Stellar 5Up - Proyecto SENA Hosting Web

## 📋 Descripción del Proyecto

Este proyecto forma parte de una actividad del SENA donde se implementa el despliegue de una plantilla web utilizando servicios gratuitos de hosting. Se utilizó la plantilla "Stellar" de HTML5 UP para demostrar el proceso completo de publicación web.

## 🌐 Sitio Web

**URL del sitio:** https://david-peralta-rd.github.io/Stellar-5Up-Host-Trabajo-Sena/

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura del sitio web
- **CSS3**: Estilos y diseño responsive
- **JavaScript**: Interactividad y efectos
- **GitHub Pages**: Plataforma de hosting gratuito
- **Git**: Control de versiones

## 📁 Estructura del Proyecto

```
Stellar-5Up-Host-Trabajo-Sena/
├── index.html          # Página principal

```

## 🔧 Instalación Local

Si deseas ejecutar este proyecto localmente:

```bash
# Clonar el repositorio
git clone https://github.com/david-peralta-rd/Stellar-5Up-Host-Trabajo-Sena.git

# Navegar al directorio
cd Stellar-5Up-Host-Trabajo-Sena

# Abrir index.html en tu navegador
# O usar un servidor local como Live Server en VS Code
```

## 📝 Personalización

### Cambiar Contenido
1. Edita `index.html` para modificar textos
2. Reemplaza imágenes en la carpeta `/images/`
3. Modifica estilos en `/assets/css/main.css`

### Colores del Tema
Los colores principales se pueden cambiar en el archivo CSS:
```css
:root {
    --primary-color: #yourcolor;
    --secondary-color: #yourcolor;
}
```
